# Tone of Voice Dermotec

**Why:** Les messages du CRM doivent sonner comme une vraie personne, pas comme un robot. Les stagiaires sont des esthéticiennes, pas des DSI. Ton chaleureux, direct, pro mais décontracté.

**How to apply:** Relire chaque template en se demandant "Est-ce que j'enverrais ça à une amie qui m'a demandé des infos ?"

## 12 RÈGLES D'OR

1. **Signer avec un prénom** (Sarah, pas "L'équipe Dermotec")
2. **1ère phrase = raison du message** (pas de warming up)
3. **1 message = 1 action** (pas de "et aussi")
4. **Détails concrets** (dates, noms, montants)
5. **Écrire comme on parle** (lire à voix haute)
6. **Poser une question** (inviter la réponse)
7. **Varier le rythme** (phrases longues + courtes)
8. **Zéro jargon Qualiopi** dans les messages clients
9. **Canal adapté** (SMS=urgent, WhatsApp=discussion, Email=détails)
10. **Fréquence respectueuse** (SMS 2-3/mois, WhatsApp 1-2/sem)
11. **Emoji avec parcimonie** (SMS: 0-1, WhatsApp: 2-3, Email: 0-1 objet)
12. **Relire avant d'envoyer**

## MOTS INTERDITS → ALTERNATIVES

| Interdit | Alternative |
|----------|-----------|
| "N'hésitez pas" | "Dites-moi si" / "On est là" |
| "Je me permets de" | "Je vous écris parce que" |

| "Nous avons le plaisir" | "Bonne nouvelle :" |
| "Restant à votre disposition" | "Appelez-moi si besoin" |

| "Afin de" | "Pour" |
| "En effet" (×15) | "oui" / "c'est vrai" / "d'ailleurs" |

## PAR CANAL

### Email
- Objet : 30-50 chars, prénom dedans, minuscules, pas de SPAM words
- Ouverture : direct au sujet, pas de "j'espère que vous allez bien"
- Corps : max 150 mots (transac), 300 (info). Listes à puces. 1 seul CTA.
- Clôture : "À bientôt !" + prénom + numéro direct

### SMS (160 chars)
- Format : "[Prénom] de Dermotec : [message] [action]. STOP"
- 1 emoji max, pas d'abréviations type texto
- Horaires : 8h-20h, jamais dimanche/fériés (loi FR)
- Max 2-3 promo/mois

### WhatsApp
- Ton le plus informel (amical-pro)
- Messages courts (3-5 lignes par bulle)
- 2-3 emoji OK
- Poser une question dans chaque message
- Voix audio = OK et apprécié pour le financement
- Pas de PDF (résumer dans le message)
