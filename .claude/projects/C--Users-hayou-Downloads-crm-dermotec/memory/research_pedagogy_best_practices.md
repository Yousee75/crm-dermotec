# Pédagogie & Apprentissage — Mars 2026

## POURQUOI 90% DE LA FORMATION ÉCHOUE
- 50% oublié en 24h, 90% en 7j (Ebbinghaus)
- Causes : passif (5% retention lecture), pas de répétition, surcharge cognitive, pas de feedback

## MODÈLE "DERMOTEC LEARNING LOOP"
1. MICRO-DÉFI (2min) — Pre-test intuitif avant la leçon (style Brilliant)
2. MICRO-LEÇON (3-5min) — Storytelling + cas réel, charge cognitive minimale
3. PRATIQUE (3-5min) — Role-play / scénario avec feedback immédiat
4. RÉFLEXION (1min) — "Qu'est-ce que j'ai appris ?" (cycle de Kolb)
5. RENFORCEMENT — Répétition espacée J+1, J+3, J+7, J+14, J+30
6. APPLICATION — Just-in-time sur le terrain (fiche réflexe mobile)
7. PARTAGE SOCIAL — Débrief équipe hebdomadaire + peer coaching

## MÉTHODE VENTE HYBRIDE "SPIN-SANDLER ESTHÉTIQUE"
- **SPIN** pour la découverte : Situation → Problème → Implication → Need-payoff
- **Sandler** pour qualification/closing : critères d'achat, engagement mutuel
- **Challenger** pour le repositionnement : enseigner, défier les croyances

## PARCOURS DE COMPÉTENCES (10 semaines)

- Niveau 2 (S3-4) : Découverte client SPIN → Badge "Maître Découverte"
