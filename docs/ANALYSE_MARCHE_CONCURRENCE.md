## 1. LE MARCHÉ

### 1.1 Formation professionnelle en France

| Indicateur | Chiffre | Source |

| CA formation pro France | **28,7 milliards EUR** (2023, +3%) | INSEE / RiseUp.ai |
| Budget France Compétences 2026 | **~12 milliards EUR** | Formation Pro Mag |

### 1.2 Niche esthétique / beauté

| Indicateur | Estimation |
|-----------|-----------|
| Centres formation esthétique FR | **3 000 - 5 000** |
| Dont certifiés Qualiopi | **~1 500 - 2 500** |
| Prix moyen formation | **700 - 2 500 EUR HT** |
| Taux financement OPCO/FT | **60-80% des inscriptions** |
| Croissance marché beauté/bien-être | **+5-8%/an** |
| Adjacent : coiffure, médico-esthétique | **+10 000 centres** |

### 1.3 Pourquoi ce marché est attractif

1. **Qualiopi obligatoire** — Tous les OF qui veulent des fonds publics DOIVENT être certifiés → besoin d'outils de gestion
2. **Financement = argent facile** — 100% finançable OPCO/FT → pas d'objection prix pour les stagiaires
3. **Digitalisation en retard** — Beaucoup de centres esthétique gèrent encore avec Excel/papier
4. **Récurrence** — Les esthéticiennes reviennent pour des formations complémentaires (upsell naturel)
5. **Matériel consommable** — Pigments, aiguilles = e-shop récurrent

## 7. VERDICT

### Forces

- **Niche vierge** : aucun CRM spécialisé esthétique en France
- **Stack moderne** : Next.js 15 + Supabase + Stripe = coût infra minimal
- **289 fichiers, 37 pages** : produit déjà très avancé
- **Gamification + IA** : aucun concurrent n'a ça
- **Marge ~90%** : modèle SaaS classique, très rentable à l'échelle
- **Marché croissant** : +7,8%/an logiciels formation

### Faiblesses

- **0 client** actuellement : product-market fit non validé
- **Seul fondateur** : risque de surcharge
- **Pas de levée de fonds** : croissance organique plus lente

### Opportunités

- **42 892 OF certifiés Qualiopi** : marché adressable énorme
- **100% finançable** : argument de vente imbattable
- **White-label** pour franchises/réseaux : CA récurrent massif
- **Expansion francophone** : Belgique, Suisse, Luxembourg, Afrique

### Conclusion

| Objectif | Réaliste ? | Comment |
|----------|-----------|---------|
| Vivre du produit (3-5K EUR/mois) | **OUI — An 1** | 50-80 clients Pro |
| Revenus confortables (10-15K EUR/mois) | **OUI — An 2** | 200 clients |
| Devenir riche (>50K EUR/mois) | **POSSIBLE — An 3** | 500+ clients ou white-label |
| ARR > 1M EUR | **AMBITIEUX — An 3-4** | Expansion + modules + scale |

**Le produit est là. Le marché est là. Il faut maintenant les 10 premiers clients.**
